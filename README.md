# Proyecto fin de ciclo

KartMania - Ricardo Andrés Santos Ren

## Descrición

O proxecto basease na creación de unha aplicación web, desarrollada no framework Ruby on Rails
a aplicación e un traballo encargado por un amigo que é dono dun karting en Galicia e que sinte 
que na sua faltan duas cousas moi importantes na actualidade, a aotomatización de procesos e o contacto
co cliente de unha maneira mais moderna e que non consuma tanto tempo.

Para estes dous aspectos o meu colega contactou conmigo para que lle fixera unha aplicación web na que os seus clientes
podan consultar información, participar en torneos, e facer reserva de algun ou varios plans do karting, xa sexa estando logueado
na aplicación ou como un cliente anónimo.

Tanto para el como para os seus empregados a aplicación servira para automatizar e facilitar procesos que na actualidade lles consuma
mais tempo do necesario, un exemplo é estar contesta chamadas para axendar reservas para os clientes, a aplicación encargarase de este
problema. Os empregados tamén teran a aplicación para apoxarse a hora de enviar correos aos clientes ou revisando que karts estan dispoñibles
a hora da reserva do cliente. 

Para os administradores tamén e moi importante facerlle chegar aos empregados a información importante, xa que o sistema de envío de correos
é o mais comun para comunicar novedades na actualidade, e tamén facilitara a descarga dos contratos dos empregados, e a xestión de calquer proceso
que sexa manexable ca aplicación.
